# HR Analytics Dashboard

## 📌 Project Overview
This project presents an **interactive HR Analytics Dashboard** built using **Power BI** to analyse employee attrition and workforce patterns.  

The goal of this project is to help organisations understand:
- Why employees are leaving  
- Which departments and roles are most affected  
- How factors like salary, age, and experience impact attrition  

This dashboard supports **data-driven HR decision making** and helps improve employee retention strategies.

## 🛠️ Tools Used
- 📊 Power BI (Dashboard & Data Visualization)  
- 🗄️ Data Modeling (Relationships, Measures, DAX)  
- 📈 Data Analysis (HR Metrics & KPIs)
